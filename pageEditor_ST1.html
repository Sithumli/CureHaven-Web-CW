Splash Page

Technical Description

The splash screen was created using HTML, CSS, and JavaScript. Key features include:

  • HTML: Used a <div> structure to organize the logo, team name, and loading animation.
  • CSS: Applied a gradient background and centered the content using Flexbox. The loading bar was styled with width and transition properties.
  • JavaScript: Implemented a timer to update the loading percentage and redirect to the home page after 4 seconds using setInterval and window.location.href.

Accessibility

Accessibility was addressed by:

  • Using semantic HTML tags for better screen reader compatibility.
  • Ensuring sufficient color contrast for text and background elements.
  • Adding alt attributes to all images for descriptive text.

Volunteer Page

Technical Description

Content Page

Technical Description

The content page provides an overview of physical health and wellness. Key features include:

  • HTML: Used semantic tags like <header>, <section>, and <article> for structure.
  • CSS: Applied a grid layout for the exercise routines section and used hover effects for interactive cards.
  • Design: Focused on a clean and modern look with consistent spacing and typography.
