Home Page

Technical Description

The home page serves as the main landing page for the website. Key features include:

  • HTML: Used semantic tags like <header>, <main>, and <footer> for structure.
  • CSS: Applied a responsive design with Flexbox and Grid for layout. Used animations for interactive elements like buttons and cards.
  • JavaScript: Added interactivity for the navigation menu and hover effects.

Accessibility

Accessibility was addressed by:

  • Using semantic HTML tags for better screen reader compatibility.
  • Ensuring sufficient color contrast for text and background elements.
  • Adding alt attributes to all images for descriptive text.

Challenges and Lessons Learned
